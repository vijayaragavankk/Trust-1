# Required assets in this /public folder

Before running `npm run build`, copy these from your original project:

## From original /public folder → /public
- poverty1.mp4  (hero background video)
- poverty.mp4   (fallback video)
- bg-home.jpg   (background for pages)
- upi-qr.png    (your UPI QR code image)
- favicon.png   (your trust logo)
